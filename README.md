# Product-Metrics-Analysis
The goal of this project is to analyse key product metrics in a hypothetical scenario and come up with recommendations for the Telco team. The company provides telecommunication services and is interested in understanding the key drivers of the businessâ€™s success.

The stakeholders are interested in the following metrics:

**MRR (Monthly Recurring Revenue)**

**Retention Rate**

**Lifetime**

**ARPPU**

**LTV**

The metrics will be calculated using MySQL DBMS, followed by explanations of the results and recommendations for the Telco team.

Dataset was taken from Kaggle:
https://www.kaggle.com/datasets/syedumeerr/telco-customer-churn/data
